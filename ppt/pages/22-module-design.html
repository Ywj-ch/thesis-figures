功能模块设计

用户界面层

Vue3 + Element Plus + Three.js

  • Views（页面视图）
    HomeView首页、SolverView求解器、LearningView学习教程
  • Components（公共组件）
    Cube3DView三维魔方、Face2DView二维展开、StepList步骤列表
  • Composables（组合式函数）
    useTheme主题切换、useCubeCustomization魔方自定义配置
  • Router（路由管理）
    Vue Router管理页面导航与懒加载

业务逻辑层

Python + FastAPI + Uvicorn

  • app.py（路由入口）
    FastAPI应用实例、CORS中间件、API路由注册
  • cube_service.py（业务聚合）
    串联识别与求解流程，协调各模块调用顺序
  • cube_image_detection.py
    调用YOLOv8模型推理，识别每面9个色块颜色
  • convert_cube_state.py
    颜色矩阵→Kociemba编码转换，校验魔方状态合法性

数据访问层

文件系统存储

  • models/（模型文件）
    存储YOLOv8训练权重（best.pt），供推理加载
  • images/（临时图片）
    存储用户上传的魔方六面照片，处理后自动清理
  • cube_results/（识别结果）
    存储识别的颜色矩阵与求解结果JSON
  • twophase/（预计算表）
    Kociemba算法剪枝表，加载后供快速求解使用
